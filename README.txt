README for MULTI_RESOLUTION_ICO
*****************************************
Para la app, se necesitan las librerías: 
- pillow

--------------------------------------
Funciona desde la consola, es demasiado sencilla para crear una interfaz gráfica. Simplemente 
en el codigo, cambiar la ruta del archivo png que se quiere convertir a .ico en diferentes resoluciones
y creara en la ruta de este codigo, el nuevo archivo .ico con las resoluciones.
